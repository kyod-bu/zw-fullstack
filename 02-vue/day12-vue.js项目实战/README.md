## 架构

**备注：** Mockjs（模拟后台请求数据）

### Vue-cli 简介

```shell
# 安装vue-cli
npm install -g @vue/cli

# 初始化
npm i -g @vue/cli-init

# 创建一个工程 demo
npm init webpack demo
...
```
